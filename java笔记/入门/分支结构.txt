
分支结构：
if分支：
if（布尔表达式）{条件满足时的执行代码；}

else if（布尔表达式）{条件满足时的执行代码；}。。。else{所有条件都不满足时的执行代码；}

注：布尔表达式的范围随便取，范围重叠都可以，反正是依次判断，只要满足当前的表达式，就执行代码后跳出所有分支，不满足则判断下一个分支；

switch分支：
若无break则会将本例及后面的case（default）全部执行出来，直到遇到有break为止；
switch（数字 | 枚举 | 字符 | 字符串）{

case 内容1：{

内容满足时执行语句；

break；

}

case 内容2：{

内容满足时执行的语句；

break；

}。。。

（default：{

都不满足时执行的语句；

break；

}）

}
